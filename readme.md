

# 得物数据爬取、分析平台

## Introduction - 介绍
该项目主要完成对得物交易数据的爬取与分析，对本地的交易数据进行分析生成一些数据分析图，能够准确的预估价格的走势。

### Summary - 概要
###1、交易记录爬虫
每天定时爬取得物交易记录，每次爬取存在记忆功能，不会存在重复爬取的现象。
###2、数据分析
每周定时对历史数据进行新的一轮分析，并生成数据分析图、数据分析报告

### Features - 特性
###1、实时性
###2、准确性
###3、个性化
